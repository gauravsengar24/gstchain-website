The GSTCHAIN Stack

Six integrated open-source projects that power the GSTCHAIN ecosystem.

📜

gst-backend

Certificate management API server

POST /auth/login → JWT token
POST /certificates → Create certificate
GET /certificates/:id → Verify
GET /events → List events
✓ All endpoints JWT-protected

Certificate Management API

A NestJS backend for issuing, managing, and verifying digital certificates on the GSTCHAIN blockchain. Features JWT authentication, Swagger documentation, MongoDB persistence, and role-based access control.

  • JWT-based authentication with bcrypt password hashing
  • Full CRUD for certificates, events, candidates, and users
  • Auto-generated timestamps and Swagger/OpenAPI docs
  • Role-based admin and user management
Source →
🔍

gst-scanner

Blockchain token indexer

syncErc20(token) → Index transfers
syncErc721(token) → Track NFTs
getHoldersOf(token) → Holder list
getInfo(token) → Token metadata
✓ Real-time with 10s polling

Blockchain Token Scanner

Real-time blockchain scanner that indexes ERC-20, ERC-721, and ERC-1155 tokens. Built with NestJS, Prisma, PostgreSQL, and ethers.js for reliable on-chain data indexing.

  • Automatic ERC-20/ERC-721/ERC-1155 token detection
  • Transfer history, balance tracking, and holder analytics
  • PostgreSQL with Prisma for type-safe queries
  • 10-second polling with exponential backoff
Source →

$ ./mainnet-archive-snapshot.sh

Blockchain Snapshots

Shell scripts to download, verify, and reassemble full and archive snapshots of the GSTCHAIN blockchain from S3, with automatic integrity verification.

  • Full and archive snapshot options
  • SHA-256 checksum verification (cross-platform)
  • Automatic part reassembly with version sorting
  • Resumable downloads with error handling
